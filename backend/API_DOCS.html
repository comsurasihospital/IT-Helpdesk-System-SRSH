🏥 IT Helpdesk API

Base URL: http://localhost:5000/api  |  v1.0.0  |  USER ADMIN SUPERVISOR

🔐 Authentication
POST /auth/line-login เข้าสู่ระบบด้วย LINE User ID

Request Body

{
  "lineUserId":       "Uxxxxxxxxx",    // จาก LIFF getProfile()
  "lineDisplayName":  "John Doe",
  "linePictureUrl":   "https://..."
}

Response (ผู้ใช้ใหม่)

{ "success": true, "data": { "isNewUser": true, "lineUserId": "Uxx" } }

Response (ผู้ใช้เดิม)

{ "success": true, "data": { "isNewUser": false, "token": "eyJ...", "user": {...} } }
POST /auth/register ลงทะเบียนผู้ใช้ใหม่

Request Body

{
  "lineUserId":    "Uxxxxxxxxx",
  "firstName":     "สมชาย",
  "lastName":      "ใจดี",
  "phone":         "081-234-5678",
  "departmentId":  2
}

Response

{ "success": true, "data": { "token": "eyJ...", "user": { "id": 10, "role": "USER" } } }
GET /auth/me ข้อมูล User ปัจจุบัน 🔒 Token

Response

{ "success": true, "data": { "id": 1, "firstName": "สมชาย", "role": "USER", "departmentName": "งานOPD" } }
GET /auth/departments รายชื่อแผนก (ใช้ตอน Register)

Response

[{ "id": 1, "name": "งานเทคโนโลยีสารสนเทศ", "code": "IT" }, ...]
🎫 Tickets
GET /tickets รายการ Ticket (USER เห็นเฉพาะตัวเอง) 🔒 Token

Query Params

?page=1&limit=20&status=OPEN&categoryId=1&search=keyword&startDate=2025-01-01&endDate=2025-12-31

Response

{
  "data": {
    "tickets": [...],
    "pagination": { "page": 1, "limit": 20, "total": 45, "totalPages": 3 }
  }
}
GET /tickets/:id รายละเอียด Ticket พร้อม comments, logs, attachments 🔒 Token
POST /tickets สร้าง Ticket ใหม่ (รองรับ multipart/form-data) 🔒 Token

Request (multipart/form-data)

categoryId:  1
title:       "HOSxP ค้างหน้าจอ"
description: "เปิดโปรแกรมแล้ว..." (อย่างน้อย 10 ตัวอักษร)
priority:    MEDIUM        (LOW | MEDIUM | HIGH | CRITICAL)
images:      [file1, file2] (optional, max 5 files)

Response 201

{ "data": { "id": 42, "ticketNo": "TK2025001", "status": "OPEN", ... } }
DELETE /tickets/:id ลบ Ticket 🔒 ADMIN
⚡ Ticket Actions
PUT /tickets/:id/accept Admin รับงาน → IN_PROGRESS 🔒 ADMIN

Response

{ "message": "รับงานเรียบร้อย", "data": { "status": "IN_PROGRESS" } }
PUT /tickets/:id/resolve Admin ปิดงาน → RESOLVED + คำนวณ SLA + ส่ง LINE 🔒 ADMIN

Request Body

{ "resolutionNote": "แก้ไขโดยติดตั้ง driver ใหม่" }
PUT /tickets/:id/cancel ยกเลิก Ticket 🔒 Token
PUT /tickets/:id/rate User ให้คะแนนความพึงพอใจ 1-5 ดาว 🔒 USER

Request Body

{ "score": 5, "note": "แก้ไขได้รวดเร็วมาก" }
POST /tickets/:id/comments เพิ่ม Comment 🔒 Token

Request Body

{ "comment": "ข้อความ", "isInternal": false }
📊 Dashboard
GET /dashboard/summary สรุปภาพรวม: จำนวน Ticket, SLA, Recent 10 🔒 ADMIN/SUP
GET /dashboard/chart/monthly กราฟ Ticket รายเดือน ?year=2025 🔒 ADMIN/SUP
GET /dashboard/chart/category กราฟตามประเภทปัญหา 🔒 ADMIN/SUP
GET /dashboard/chart/department กราฟตามแผนก 🔒 ADMIN/SUP
📈 Reports
GET /reports/tickets รายงาน Ticket แบบ Filter (startDate, endDate, status, ...) 🔒 ADMIN/SUP
GET /reports/sla รายงาน SLA Performance 🔒 ADMIN/SUP
GET /reports/admin-performance ประสิทธิภาพทีม IT 🔒 ADMIN/SUP
👥 User Management
GET /users รายชื่อผู้ใช้ทั้งหมด ?role=ADMIN&search=ชื่อ 🔒 ADMIN/SUP
PUT /users/:id แก้ไขข้อมูลผู้ใช้ 🔒 ADMIN
DELETE /users/:id ระงับบัญชี (soft delete) 🔒 ADMIN
